# Free Food Stockpile

Marketing + application website for the Free Food Stockpile program — free groceries and
ready-to-eat meal support for approved individuals and families in Hudson Valley, NY.

- `index.html` — landing page (hero, how it works, eligibility, apply CTA + QR, contact)
- `apply.html` — 2-step application form (applicant + household + eligibility/health)
- `assets/` — CSS, JS, food photos (AI-generated), logo, and the Scan-to-Apply QR code

Domain: freefoodstockpile.org · Built by BH Web Solutions.
